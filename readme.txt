This free pace calculator computes pace, distance and time by setting values of two variables. You can use it for training purposes.

[https://www.calculator.io/pace-calculator/](https://www.calculator.io/pace-calculator/)

== Usage ==

Add the Pace Calculator shortcode to your page, post or sidebar:

`[ci_pace_calculator]`

Add the following code to your template where you would like the Pace Calculator to appear:

`<?php display_ci_pace_calculator(); ?>`

== Installation ==

1. Upload the Pace Calculator /ci_pace_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Pace Calculator plugin through the "Plugins" menu in WordPress.
